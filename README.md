📊 Sales Prediction using Machine Learning
This project focuses on predicting product sales based on advertising expenditure using machine learning techniques in Python.

🚀 Project Overview
The objective of this project is to analyze the impact of different advertising platforms and build a predictive model that forecasts sales. The project also includes audience segmentation to understand different customer groups.

🔍 Key Features
🔹 Data Cleaning and Preprocessing
🔹 Exploratory Data Analysis (EDA)
🔹 Data Visualization using Matplotlib and Seaborn
🔹 Sales Prediction using Linear Regression
🔹 Model Evaluation (R² Score, Mean Squared Error)
🔹 Audience Segmentation using K-Means Clustering
🔹 Business Insights for advertising strategy optimization

📈 Technologies Used
✔ Python
✔ Pandas
✔ NumPy
✔ Matplotlib
✔ Seaborn
✔ Scikit-learn
✔ Jupyter Notebook
